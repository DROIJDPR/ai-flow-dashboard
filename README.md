AIFlow Dashboard

Modern SaaS dashboard built with SvelteKit and TypeScript.

Live Demo

https://ai-flow-dashboard.vercel.app/dashboard

Features

* Authentication UI
* Dashboard Analytics
* Users Management
* AI Agents Monitoring
* Settings Module
* Dark / Light Theme
* Responsive Layout

Tech Stack

* SvelteKit
* TypeScript
* CSS
* Vite

Installation

npm install
npm run dev

Production Build 

npm run build
